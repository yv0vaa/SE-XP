Руководство пользователя

Регистрация и вход

Регистрация нового пользователя

  1. Перейдите на страницу регистрации

  2. Заполните форму:

    • Имя пользователя (username)

    • Имя (first name)

    • Фамилия (last name)

    • Email

    • Пароль

    • Роль (студент или преподаватель)

  3. Нажмите «Зарегистрироваться»

После успешной регистрации вы будете автоматически авторизованы.

Вход в систему

  1. Перейдите на страницу входа

  2. Введите имя пользователя и пароль

  3. Нажмите «Войти»

Работа для студентов

Просмотр курсов и заданий

На главной странице студента (Dashboard) отображаются:

  • Все курсы, на которые студент записан

  • Статистика по всем курсам

При переходе в конкретный курс отображаются:

  • Все домашние задания курса

  • Статус сдачи каждого задания

  • Просроченные задания (отмечены красным)

  • Оценки за проверенные работы

Примечание

Студент видит только те курсы, на которые его записал преподаватель!

Отправка решения

  1. Откройте курс из списка «Мои курсы»

  2. Выберите домашнее задание из списка

  3. Нажмите на название задания

  4. Прочитайте описание и требования

  5. Загрузите файл с решением

  6. Нажмите «Отправить работу»

Примечание

Каждое задание можно отправить только один раз!

Просмотр результатов

Мои работы (/student/submissions/):

  • Список всех отправленных работ

  • Статус проверки каждой работы

  • Полученные оценки и комментарии

Мои оценки (/student/grades/):

  • Таблица оценок по всем курсам

  • Оценки организованы по курсам и заданиям

  • Видно, какие задания не сданы, на проверке или оценены

Работа для преподавателей

Создание курса

  1. На главной странице (Dashboard) нажмите «Создать курс»

  2. Заполните форму:

    • Название курса

    • Описание курса

  3. Нажмите «Создать курс»

Примечание

После создания курса вы будете автоматически назначены его преподавателем.

Запись студентов на курс

  1. Откройте курс из списка «Мои курсы»

  2. Нажмите кнопку «Студенты»

  3. Отметьте студентов, которых хотите записать на курс

  4. Нажмите «Сохранить список студентов»

Создание домашнего задания

  1. Откройте курс из списка «Мои курсы»

  2. Нажмите «Создать задание»

  3. Заполните форму:

    • Название задания

    • Описание задания

    • Срок сдачи (дата и время)

  4. Нажмите «Создать задание»

Примечание

Задание будет доступно только студентам, записанным на этот курс.

Просмотр работ студентов

Есть два способа просмотра работ:

По конкретному заданию:

  1. Откройте курс из списка «Мои курсы»

  2. Выберите задание из списка

  3. Нажмите «Работы»

  4. Увидите список всех отправленных работ по этому заданию

Все работы по всем курсам:

  1. В навигационном меню нажмите «Все работы» (/teacher/submissions/)

  2. Можно отфильтровать по статусу:

    • Все работы

    • Непроверенные (?status=pending)

    • Проверенные (?status=graded)

Проверка и выставление оценки

  1. Выберите работу студента из списка работ

  2. Нажмите «Проверить»

  3. Скачайте и просмотрите файл с решением (кнопка «Скачать файл»)

  4. Введите оценку (число от 0 до 100)

  5. Добавьте отзыв/комментарий (опционально)

  6. Нажмите «Сохранить оценку»

Редактирование курса

  1. Откройте курс из списка «Мои курсы»

  2. Нажмите «Редактировать курс»

  3. Внесите изменения в название или описание

  4. Нажмите «Сохранить изменения»

Редактирование задания

  1. Откройте курс из списка «Мои курсы»

  2. Найдите задание в списке

  3. Нажмите «Редактировать» рядом с заданием

  4. Внесите изменения

  5. Нажмите «Сохранить изменения»

Примечание

Преподаватель может управлять только своими курсами и заданиями.

Предупреждение

При удалении курса через админ-панель будут удалены все связанные с ним задания и работы студентов!
